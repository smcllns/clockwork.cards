## -- MATH --

In binary, which computers use, 9 is 1001. "Binary" just means using base 2. Humans mostly write numbers in base 10. "Base 10" means our numbers are organized in powers of ten: 10->100->1000. Binary is exactly the same idea, but instead of powers of 10, it uses powers of 2 like 2->4->8->16. We can do all the same calculations with either! Weird right? But the math checks out:

In base 2, 1001 means: 1 eight, 0 fours, 0 twos, and 1 one, which adds up to 9.
1001sub(2) = (1 x 2^3) + (0 x 2^2) + (0 x 2^1) + (1 x 2^0) = 8 + 0 + 0 + 1 = 9

In base 10, 9 means: no hundreds and no tens and nine ones, which adds up to 9.
9sub(10) = (0 x 10^3) + (0 x 10^2) + (0 x 10^1) + (9 x 10^0) = 0 + 0 + 0 + 9 = 9
